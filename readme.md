## **2주차: 개발 환경 설정 및 테스트**

- **환경 구축**: VS Code, JDK, 스프링 부트 확장 프로그램을 설치하고 설정을 완료함.
- **프로젝트 생성**: Spring Initializr로 Maven 기반 프로젝트를 생성하고 필요한 의존성을 추가함.
- **페이지 실행**: `index.html`, `hello.html`을 생성하고 컨트롤러를 통해 웹 페이지를 실행함.
- **버전 관리**: Git을 설치하고 GitHub 원격 저장소에 프로젝트를 연동함.

---

## **3주차: 포트폴리오 작성하기 (프론트엔드)**

- **템플릿 적용**: 부트스트랩5 기반 포트폴리오 템플릿을 다운로드하여 프로젝트에 적용함.
- **화면 수정**: 메인 페이지의 내용을 개인 정보에 맞게 수정하고 프로필 사진을 교체함.
- **페이지 추가**: 상세 소개 페이지(`about_detailed.html`)를 만들고 컨트롤러에 URL 맵핑을 추가함.

---

## **4주차: 데이터베이스 연동 및 테스트**

- **DB 연동**: MySQL을 설치하고 `application.properties`에 연결 정보를 설정함.
- **구조화**: MVC 패턴에 따라 프로젝트 패키지 구조를 세분화함.
- **연동 테스트**: Entity, Repository, Service, Controller를 구현하여 DB 데이터를 웹 페이지에 출력함.

---

## **5주차: 블로그 게시판 - 1 (조회 및 글쓰기)**

- **조회(Read)**: 모든 게시글을 가져와 목록 페이지에 출력하는 기능을 구현함.
- **작성(Create)**: 새 글을 입력받는 폼을 만들고 `POST` 요청을 처리하여 DB에 저장함.
- **DTO 사용**: 계층 간 데이터 전송을 위해 DTO 객체를 활용함.

---

## **6주차: 블로그 게시판 - 2 (수정 및 삭제)**

- **수정(Update)**: 수정 폼을 만들고 `PUT` 요청을 통해 게시글 내용을 업데이트함.
- **삭제(Delete)**: `DELETE` 요청을 통해 특정 게시글을 삭제하는 기능을 구현함.
- **예외 처리**: 존재하지 않는 게시글 접근 시 에러 페이지로 이동시키는 로직을 추가함.
